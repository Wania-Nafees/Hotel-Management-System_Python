# Hotel-Management-System_Python

A desktop-based Hotel Management System built using Python and Tkinter to manage hotel rooms, guest check-ins/check-outs, and billing. This project was developed as a personal learning project to strengthen core programming and GUI concepts.

-Features:
Admin Login & Registration system
Secure Admin Dashboard
Add new hotel rooms dynamically
View real-time room availability
Guest Check-in & Check-out
Automatic bill calculation
Booking history with check-in & check-out date/time
Logout functionality
Professional, hotel-style GUI theme
All data stored in TXT files (no database)

-Data Storage:
admins.txt → Admin credentials
rooms.txt → Room details & availability
bookings.txt → Complete booking history (check-in, check-out, bill)

-Concepts Used:
Python programming
Tkinter GUI
File handling (.txt files)
Data Structures (dictionary, list)
Date & time handling (datetime)
Event-driven programming
Real-world system design
